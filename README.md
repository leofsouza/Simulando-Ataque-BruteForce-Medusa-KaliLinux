# Simulando Ataques de Brute Force

Este repositório contém um conjunto de ferramentas para simular ataques de brute force em senhas, demonstrando a vulnerabilidade das senhas fracas e a importância de implementações de segurança eficazes.

## Funcionalidades

- Simulação de ataques de força bruta.
- Relatórios de eficiência do ataque.
- Análises de senhas testadas.

## Uso

Para executar a simulação, utilize o seguinte comando:

```bash
python brute_force_simulator.py
```
